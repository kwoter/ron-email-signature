# Ron Atkinson — Email Signature

Premium HTML email signature for Ron Atkinson Consultancy Limited, featuring
Add-On Exchange + Claims Exchange as headline brands and kwoter, kompare,
kompare calls and greener travel as partner brands.

- `signature.html` — production signature (images served via jsDelivr CDN)
- `assets/` — optimised retina logo PNGs (served by jsDelivr)
- `src/` — build + render scripts

Logos are served from:
`https://cdn.jsdelivr.net/gh/kwoter/ron-email-signature@main/assets/`

See INSTALL.md for Outlook setup.
