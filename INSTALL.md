# Installing Ron's signature in Outlook

Live copy-paste page: **https://kwoter.github.io/ron-email-signature/**
All logos are served from a CDN, so they display for every recipient — nothing is attached to the email.

## Easiest method (all Outlook versions, Outlook for Mac, and Outlook on the web)
1. Open **https://kwoter.github.io/ron-email-signature/** in a web browser (Chrome, Edge or Safari).
2. Click on the signature, press **Ctrl+A** (⌘A on Mac) to select all, then **Ctrl+C** (⌘C) to copy.
3. In Outlook open the signature editor:
   - Classic Outlook (Windows): **File → Options → Mail → Signatures**
   - New Outlook / Outlook on the web: **Settings → Mail → Compose and reply → Email signature**
   - Outlook for Mac: **Outlook → Settings → Signatures**
4. Create a new signature named "Ron", click into the edit box and **paste (Ctrl+V / ⌘V)**.
5. Set it as the default for **New messages** and **Replies/forwards**, then save.

## Notes
- Paste normally — do **not** use "paste as plain text", or the logos and links will be stripped.
- If a logo looks momentarily missing in the editor, it is just loading from the CDN; it will display in sent mail.
